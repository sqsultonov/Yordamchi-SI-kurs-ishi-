# Yordamchi-SI-kurs-ishi-
"Yordamchi SI" kurs ishim uchun sun’iy intellektga asoslangan dasturiy vosita. U foydalanuvchining ovozini matnga aylantiradi, matnni ovozda o‘qiydi, kamera yoki rasm orqali obyektlarni aniqlaydi hamda rasmdagi matnni o‘qiy oladi. Dastur oflayn rejimda ishlaydi va Python asosida ishlab chiqilgan.
